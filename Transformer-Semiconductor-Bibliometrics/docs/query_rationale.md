# 检索式设计思路说明 (Query Rationale)

> 创建日期：2026-05-01
> 版本：v1.0

---

## 1. 检索目标

系统检索 2015-2025 年间 Transformer 架构在半导体制造领域应用的学术文献，支撑文献计量分析。

---

## 2. 检索词设计逻辑

### 2.1 三层结构

检索式采用「对象词 + 场景词 - 排除词」三层结构：

| 层次 | 作用 | 设计原则 |
|---|---|---|
| **对象词** | 锁定技术对象 | 覆盖 Transformer 主流变体，避免遗漏 |
| **场景词** | 限定应用领域 | 使用上位词而非具体应用词 |
| **排除词** | 提高查准率 | 排除同名异义和无关场景 |

### 2.2 对象词选择

| 词项 | 选择理由 |
|---|---|
| Transformer | 核心架构名称 |
| Vision Transformer | 图像领域主流变体 |
| ViT | 常用缩写 |
| Swin Transformer | 高影响力变体（ICCV 2021）|
| Autoformer | 时序预测专用变体 |
| Informer | 长序列预测变体 |
| PatchTST | 时序预测新变体（2023）|

**覆盖策略**：不限定于单一变体，避免遗漏新兴架构。

### 2.3 场景词选择

| 词项 | 选择理由 |
|---|---|
| semiconductor | 领域上位词，覆盖全场景 |
| wafer | 晶圆制造核心词 |
| integrated circuit | IC 制造通用词 |
| lithography | 光刻工艺专用词 |
| semiconductor packaging | 封装工艺 |
| wafer fabrication | 晶圆制造 |
| chip manufacturing | 芯片制造 |

**设计原则**：使用上位词而非具体应用词（如 defect detection），提高查全率。

### 2.4 排除词选择

| 词项 | 排除理由 |
|---|---|
| power transformer | 同名异义（电力变压器）|
| voltage | 电力系统噪声 |
| current transformer | 电力系统噪声 |
| medical image | 医学影像领域噪声 |
| NLP | 自然语言处理领域噪声 |
| natural language | NLP 噪声 |
| traffic | 交通预测领域噪声 |
| power load | 电力负荷预测噪声 |
| smart grid | 智能电网领域噪声 |
| distribution network | 配电网领域噪声 |
| financial | 金融预测领域噪声 |

**排除策略**：基于 v1 检索结果人工核查，识别高频噪声源后针对性排除。

## 3. 检索式迭代过程

### 3.1 v0 → v1：放宽场景

| 变化 | v0 | v1 |
|---|---|---|
| 对象词 | 3 个 | 7 个（新增 Swin/Autoformer/Informer/PatchTST）|
| 场景词 | 具体应用词 | 上位词 |
| 结果数 | 105 | 512 |

**原因**：v0 过度依赖具体应用词（defect detection），查全率过低。

### 3.2 v1 → v2-final：补充排除词

| 变化 | v1 | v2-final |
|---|---|---|
| 排除词 | 6 个 | 11 个 |
| 结果数 | 512 | 643 |

**原因**：v1 包含电力系统、交通预测、金融等噪声文献。

## 4. 质量控制措施

### 4.1 NOT 误杀检查

随机抽取 10 条结果，人工核查排除词是否误删相关文献。

| 检查项 | 结果 |
|---|---|---|
| 抽样数 | 10 |
| 误杀数 | 0 |
| 结论 | 排除词未误删 |

### 4.2 查准率评估

| 检查项 | 结果 |
|---|---|---|
| 抽样数 | 10 |
| 高度相关 | 9 |
| 边缘相关 | 1 |
| 不相关 | 0 |
| 查准率估计 | 90%+ |

### 4.3 查全率评估

| 检查项 | 结果 |
|---|---|---|
| Transformer 主流变体 | 已覆盖 |
| 半导体核心场景 | 已覆盖 |
| 估计查全率 | 高 |

---

## 5. 检索式局限性

| 局限 | 说明 | 影响 |
|---|---|---|
| 未覆盖 CNKI | 仅检索 WOS | 可能遗漏国内中文文献 |
| 未覆盖新兴变体 | Transformer 变体持续涌现 | 需定期更新 |
| 排除词可能过严 | 后续新兴应用可能被误排 | 需人工复核 |
