> 基于 PRISMA 声明框架
> 适用范围：Transformer 在半导体制造领域的文献计量分析
> 创建日期：2026-05-01

| 阶段 | 操作 | 纳入 | 排除 | 排除标准 |

## 2. 初筛标准（标题 + 摘要）

### 2.1 纳入标准

| 编号 | 标准 | 说明 |
|---|---|---|
| I1 | 研究对象为 Transformer 或其变体 | 包括 ViT, Swin, Autoformer, Informer, PatchTST 等 |
| I2 | 应用场景为半导体制造相关 | 包括晶圆检测、光刻、封装、IC 设计、缺陷检测等 |
| I3 | 文献类型为 Article, Review, Proceedings Paper | 排除 Editorial, Letter, News 等 |
| I4 | 发表时间为 2015-2025 | 时间范围限定 |
| I5 | 语言为 English | 语言限定 |

### 2.2 排除标准

| 编号 | 标准 | 排除原因编码 |
|---|---|---|
| E1 | 研究对象为电力变压器（power transformer） | EX-PWR |
| E2 | 应用场景为医疗影像、NLP、交通预测、金融等 | EX-DOMAIN |
| E3 | 文献类型为 Editorial, Letter, News, Correction | EX-TYPE |
| E4 | 重复发表（基于 DOI/UT 去重） | EX-DUP |
| E5 | 摘要缺失且标题无法判断相关性 | EX-NOINFO |

## 3. 复筛标准（全文核查）

### 3.1 纳入标准

| 编号 | 标准 | 说明 |
|---|---|---|
| F1 | 方法论明确 | 使用 Transformer 架构进行建模或分析 |
| F2 | 数据来源可追溯 | 实验数据或仿真数据有明确来源 |
| F3 | 结果可理解 | 结论清晰，可提取关键信息 |

### 3.2 排除标准

| 编号 | 标准 | 排除原因编码 |
|---|---|---|
| EF1 | 仅提及 Transformer 但未作为核心方法 | EX-MENTION |
| EF2 | 数据来源不明或不可复现 | EX-NODATA |
| EF3 | 全文无法获取 | EX-NOACCESS |
| EF4 | 非英语文献（WOS 标注错误） | EX-LANG |
