# 项目流程

> 基于 Transformer 的半导体制造关键技术研究热点与发展趋势——文献计量分析（2015-2025）

---

## 1. 选题与检索

1. 查阅课程课件 L1-L9，确定选题方向：Transformer × 半导体制造交叉领域
2. 登录 WOS → 构建检索式，包含 6 种 Transformer 变体 + 半导体场景词 + 11 类排除词
3. 三轮迭代：v0（105条）→ v1（512条）→ v2-final（**643条**）
4. 导出 Plain Text 格式（全记录 + 引用参考文献），分两批（500+143）
5. 对照课标完成 PRISMA 筛选流程图 + 查新报告

## 2. 数据清洗与质控

1. CiteSpace 去重（基于 UT + DOI 双重验证）→ **零重复**
2. 字段完整率统计（TI/AU/C1/PY/AB/CR 完整率 99%+）
3. 编写 52 字段字典 + 数据质量报告（评级 **A**）
4. 初筛 → 复筛，建立排除原因编码表

## 3. 计量分析

1. **CiteSpace**（核心工具）：关键词共现、共被引网络、突现检测、聚类与时间线
2. 参数：时间切片 2015-2025（Slice=1），g-index（k=25），Pathfinder 裁剪，LLR 聚类标签
3. 产出指标：文献总数 643，总被引 5,974，h-index 37，作者数 2,517

## 4. 结果分析

### 4.1 发文趋势

- 萌芽期（2015-2019）：年均 31 篇，平稳
- 加速期（2020-2023）：年均 52 篇，ViT 提出后稳步增长
- **爆发期（2024-2025）**：年均 134 篇，大模型热潮驱动

### 4.2 国家与机构

- **中国主导**：发文量占 40.4%（260 篇）
- 美国（12.4%）、韩国（8.9%）紧随其后
- 核心机构：东南大学（14）、中科院（12）、电子科大（12）

### 4.3 关键词共现与聚类

- 识别研究热点：缺陷检测、时序预测、图像识别
- LLR 聚类标签展示研究主题分布

### 4.4 共被引网络

- 揭示知识基础：纳米生物传感器、医学影像迁移、器件建模
- 高被引文献：Zheng 2005、Cui 2001 等

### 4.5 突现检测

| 关键词 | 突现区间 | 强度 |
|---|---|---|
| transformers | 2023-2025 | 2 |
| deep learning | 2024-2025 | 2 |
| vision transformer | 2025 | 1 |

**结论**：Transformer 架构从 2023 年开始大规模渗透半导体制造领域

## 6. 项目复盘

### 做得好的

- 检索式三轮迭代，从 105→512→643 条，逻辑清晰记录完整
- 文档体系覆盖了课程要求的全部规范性文档

### 踩过的坑

- GitHub Pages 部署时 `git add -A` 混入 node_modules，改为精确添加文件
- Tailwind CSS v4 需要 `@tailwindcss/vite`，旧版 PostCSS 配置会报错
- 分支切换会破坏 node_modules，每次切回需重装
- lucide-react 无 `Github` 图标，用了 `ExternalLink` 替代
