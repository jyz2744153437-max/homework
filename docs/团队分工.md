## 详细分工

### 成员1 高宇翔（数据与图谱主负责人）

| 任务项 | 具体内容 |
|---|---|
| 选题确定 | 针对 Transformer 架构在半导体制造领域的交叉研究现状进行文献调研，结合 WoS 数据库特点锁定可行选题方向，撰写选题背景与研究意义 |
| 检索式搭建 | 独立完成 WoS 检索式初始设计，在成员2协助下经历 v0（105条）→ v1（512条）→ v2-Final（643条）三轮迭代，涵盖 6 种 Transformer 变体、7 类半导体术语，精准排除 11 类噪声干扰 |
| WoS 文献检索与导出 | 在 Web of Science Core Collection 执行最终检索式，分两批导出 643 篇文献的 Plain Text 全记录（含 52 个标引字段），去重后交付数据文件 |
| CiteSpace 图谱制作 | 独立操作 CiteSpace 6.4.1，制作 6 类核心图谱：关键词共现网络（含 LLR 聚类）、关键词聚类图（Cluster View）、文献共被引网络、共被引聚类图、突现检测时间线（Kleinberg 算法）、聚类时间线演化图（Timeline View） |
| 图谱参数调优 | 配合成员3完成参数设置与验证：g-index（k=25）、Pathfinder + Pruning sliced networks 裁剪、LLR 聚类标签提取，核验 Modularity Q 与 Mean Silhouette 指标达标 |
| 图谱导出 | 统一导出为 PNG 格式（高分辨率），按规范命名，交付至项目 outputs 目录 |

### 成员2 纪彦泽（项目推进与成果负责人）

| 任务项 | 具体内容 |
|---|---|
| 全程项目统筹 | 根据项目里程碑（M1 数据检索 → M2 计量分析 → M3 终稿交付），把控各阶段进度与交付节点，协调成员间任务衔接，确保在规定时间内完成全部产出 |
| 检索式设计参与 | 参与检索式三轮迭代讨论，协助成员1分析各版本检索结果的噪声来源与遗漏问题，提出排除词策略优化建议，最终锁定 643 篇高质量文献集 |
| WoS 检索协助 | 协助成员1完成 Web of Science 数据库检索操作，验证检索结果数量与导出完整性，核对两批数据的一致性 |
| 数据质控文档编写 | 主导撰写 52 字段完整数据字典、数据质量评级报告（A 级）、PRISMA 标准筛选流程图（Mermaid 格式）、筛选排除原因编码表、数据清洗规则说明等全套质控文档 |
| GitHub 仓库建设 | 仓库架构设计（目录结构、文档分层）、README 与 PROGRESS.md 持续维护更新、v1.0 Release 发布管理、gh-pages 分支独立部署、Git 历史清理与标签管理 |
| 项目展示网页 | 基于 HTML5 + Tailwind CSS + Lucide 图标库 + ECharts 5.5.0 构建 30 页交互式学术幻灯片，集成发文趋势双轴图表（柱状图+增长率折线，含 ViT 标注与三阶段划分线），适配桌面全屏翻页与手机竖向滚动双模式 |
| 汇报 PPT 制作 | 整合 CiteSpace 六类图谱与核心计量指标，按研究背景→数据方法→结果分析→深度讨论→结论指引的学术逻辑编排演示文稿 |
| 辅助分析脚本 | 编写 Python 突现检测脚本（Kleinberg 算法）、Node.js 网络指标计算脚本、Python 文献计量管道脚本（run_pipeline.py） |
| 查新与对标 | 独立完成查新报告（确认无直接竞争综述），完成对标参照组分析（五维度对比），检索式变更日志全程记录 |
| 人机协作调度 | 负责人机协作整体调度。使用 Claude Code（接入 DeepSeek-v4-pro API）进行项目总成与推进，Gemini 3 Pro 进行 HTML 主体开发，DeepSeek-v4-pro / GLM-5.1 / GPT-5.4 负责代码生成与辅助分析，NanoBanana 2 与 GPT Image 2 参与图片生成。人工负责选题方向、质量把关与多轮迭代调校 |

### 成员3 韩建财（数据质控与图谱协理）

| 任务项 | 具体内容 |
|---|---|
| 文献筛选 | 深度参与 PRISMA 标准筛选流程，对照筛选规则文档逐篇审核初筛文献，标记排除原因编码，完整记录筛选过程，确保纳入文献的学科相关性 |
| 数据清洗 | 协助完成 WoS 导出数据的去重核验（CiteSpace 去重 + UT/DOI 双重交叉比对）、字段规范化处理，配合成员2完成 52 字段数据字典中各字段含义与完整率的统计与标注 |
| 数据质量报告 | 参与数据质量评估工作，协助统计核心字段完整率（TI/AU/AB/CR 等），核对年度发文分布与期刊分布数据，配合完成 A 级质量评级 |
| 图谱参数核验 | 协助成员1完成 CiteSpace 关键参数的设置与测试：验证不同 g-index k 值（20/25/30）对网络结构的影响、Pathfinder 裁剪效果对比、LLR 聚类标签的合理性判断 |
| 网络指标验证 | 核验图谱输出的 Modularity Q 值（> 0.3 合格）与 Mean Silhouette 值（> 0.7 合格），如不达标则协助调整参数重新运行至合格为止 |
| 辅助文献整理 | 协助从最终图谱中提取 Top 关键词、Top 被引文献、核心作者/机构等关键信息，为成员4的文献分析提供数据支撑 |

### 成员4 王贺东（文献分析与成果协理）

| 任务项 | 具体内容 |
|---|---|
| 核心文献精读 | 深度参与共被引网络中高被引核心文献（Top 20）的全文阅读与内容摘要，梳理 Wei (2022)、Fan (2024)、Chen (2023) 等在晶圆缺陷检测中应用 Transformer 的核心文献，归纳知识传承关系 |
| 知识基础梳理 | 协助从共被引聚类结果中归纳三大知识群组：纳米级基础理论群、医学影像技术跨域溢出群、基础架构破局点群，撰写知识基础分析段落 |
| 研究热点提炼 | 协助从关键词共现网络与聚类结果中提炼三大核心研究主题（晶圆缺陷检测、工业时序预测、视觉图像识别），绘制研究热点结构表，标注各主题代表关键词 |
| 三阶段演进分析 | 协助根据发文趋势数据划分三阶段（萌芽期 2015-2019 / 加速期 2020-2023 / 爆发期 2024-2025），提炼各阶段特征与关键拐点事件（如 2020 年 ViT 提出），撰写阶段演进说明 |
| PPT 内容架构 | 协助成员2设计汇报 PPT 的逻辑框架（六大部分 30 页），编排各章节内容板块，审核图文匹配与数据准确性 |
| 展示素材整理 | 协助收集整理网页所需的 CiteSpace 图谱 PNG 文件、核心文献引文信息、指标统计数据与参考文献列表，确保网页数据与报告一致 |
