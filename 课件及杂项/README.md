## 关键收获

### 课程三大里程碑
- **M1**（第4周）：数据与检索方案 — 20分
- **M2**（第10周）：计量分析产出 — 20分
- **M3**（第15周）：终稿与可复现 Release — 20分

### 综述写作五步法（L17-18 核心）
1. **定问题** — 对象 + 场景 + 技术维度
2. **定数据** — 检索式 + 数据库 + 年份
3. **定图谱** — 关键词/共被引/突现/结构变异
4. **定章节** — 聚类标签 → 技术叙事线
5. **定判断** — milestone 论文 + 关键指标

### 三图互证框架
- 关键词共现 → 看"热点分散度"
- 共被引网络 → 看"基础集中度"
- 结构变异 → 看"领域是否重组"

### 范本论文结构（对标 Nanoscale 2025）
```
材料 → 器件 → 算法 → 系统 → 趋势 → 挑战 → 结论
```
### 关键技巧
- 异常点解读四步法：外部事件 / 数据库滞后 / 统计口径变化 / 领域转向
- Milestone 四维度筛选：高被引 / 突现强度 / 中介中心性 / 聚类位置
- 小而交叉领域用"主题+施引"双层数据集设计
